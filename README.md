# 🎬 Movie Recommendation System

A Machine Learning based movie recommendation system that suggests movies similar to the one selected by the user.

The system uses similarity algorithms and movie metadata to recommend relevant movies.

---

## 🚀 Features

* Movie recommendation based on similarity
* Uses TMDB API for movie posters
* Interactive UI using Streamlit
* Fast recommendation results

---

## 🧠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* TMDB API

---

## 📂 Project Structure

movie-recommendation-system

app.py → main application
movies.pkl → movie dataset
similarity.pkl → similarity matrix
requirements.txt → dependencies
README.md → project documentation

---

## ⚙️ Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/Movie-Recommender-system-using-ML.git

Move to the project folder:

cd Movie-Recommender-system-using-ML

Install dependencies:

pip install -r requirements.txt

---

## ▶️ Run the Project

Run the Streamlit app:

streamlit run app.py

---

## 🔑 API Setup

Create a `.env` file in the project directory and add your TMDB API key:

API_KEY=your_api_key_here

---

## 📊 Dataset

The movie dataset is taken from TMDB.

Due to GitHub file size limits, large dataset files are not included in the repository.
